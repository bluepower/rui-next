# useUnmount

A hook that executes a function before the component unmount.

```ts
useUnmount(fn: () => void);
```

## API

### useUnmount - Params

| Property | Description                        | Type         | Default |
| -------- | ---------------------------------- | ------------ | ------- |
| fn       | The function needs to be executed. | `() => void` | -       |
