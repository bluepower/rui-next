# Getting Started

> react-hooks `demo+docs` playground based on [vitepress-rc](https://github.com/cvnine/vitepress-rc).

## Project setup

Install dependencies

```bash
$ npm install
```

Start the dev server for docs-dev

```bash
$ npm start
```

Build documentation

```bash
$ npm run docs-build
```
