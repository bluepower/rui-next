# Getting Started

> react-hooks `demo+docs` playground based on [vitepress-rc](https://github.com/cvnine/vitepress-rc).

## Project setup

Install dependencies

```bash
$ npm install
```

Prepare for docs-dev

```bash
$ npm run dev
```

Start the dev server for [docs-dev](http://localhost:3000/rui-next/)

```bash
$ npm start
```

Build lib

```bash
$ npm run build-lib
```

Build documentation

```bash
$ npm run docs-build
```
