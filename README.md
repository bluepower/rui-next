## Project setup

Install dependencies

```bash
$ npm install
```

Start the dev server for [docs-dev](http://localhost:3000/rui-next/docs/)

```bash
$ npm start
```

Build lib

```bash
$ npm run build-lib
```

Build documentation

```bash
$ npm run build-docs
```
